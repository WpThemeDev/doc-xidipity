Description:Early bound template installation directions.

 

An early bound template is a template which requires updates prior to adding it to your content. Although the add template tool ( post_add ) permits last minute changes, this is not an editor and may prove to be inadequate for this purpose. doc.Xidipity(.com) provides a sandbox editor for this purpose or most other HTML editors will suffice.

 

Templates requiring early configuration will contain highlighted markers surrounded by "#" symbols. These markers provide a visual clue identifying items to be updated. For example, the template "#ICON#" marker will need to be replaced with either a font awesome icon element, Google icon element, or other suitable image element. To illustrate; to display the ( ​ ) font awesome icon in a template with an "#ICON#" marker, replace the marker with the following.

 

<i class="far fa-file-alt">&#x200B;</i>"

 

Steps

  1. Identify desired template
  2. Highlight template
  3. Copy to clipboard
  4. Optionally
    1. Paste into external editor
    2. Configure as desired
    3. Highlight updated template
    4. Copy updated template to clipboard
  5. Click the location where template will be applied
  6. Click the add template ( post_add ) toolbar icon
  7. Paste template into the tool
  8. Perform final configuration if required
  9. Click OK button
