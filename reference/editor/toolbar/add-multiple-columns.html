Selection Type:Dropdown Menu

 

Selection Function
Auto 2 Column Automatically create 2 equally balanced columns
Auto 3 Column Automatically create 3 equally balanced columns
Auto 4 Column Automatically create 4 equally balanced columns
Fixed 2 Column Create 2 columns with content placement defined by author
Fixed 3 Column Create 3 columns with content placement defined by author
Fixed 4 Column Create 4 columns with content placement defined by author
