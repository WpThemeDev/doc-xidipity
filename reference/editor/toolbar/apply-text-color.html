Toolbar Button

 

This toolbar selection presents 90 colors which may be applied as a foreground or background color to selected text. In the circumstance where the desire is to use a custom color, key in the hex code into the lower left input and then click the swatch to the left to view. New to version 1.01 is support for 148 named colors. Type the desired color into the input and click the swatch to view, OK to apply.
