excerpt

 

editor view

 

Use the power of creative composition to entice visitors to want to read more by adding custom excerpts to your blog content. Xidipity's cleverly designed excerpt is easy to use and provides opportunities to entice readers with compelling content.

By default, WordPress will use the first twenty five characters of your post to create an excerpt for the blog summary page. This is an imperfect solution as it includes all content types such as headers (h1-h6) which may not produce the desired result. The best solution is to create your own excerpt. It should be noted WordPress does not recognize every text style or other embellishments like font awesome icons in the excerpt. Unicode characters and emojis are acceptable.

acronym

 

When abbreviating a term, use the full term the first time you use it, followed immediately by the abbreviation in parentheses. The abbreviation for Cascading Style Sheets (css) is a good example. To use this selection highlight the desired content and select acronym from the menu.
