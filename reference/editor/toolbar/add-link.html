Selection Type:Button

 

Selection Function
default add link to selected element
edit/delete edit or remove a link from an established link

 

default

 

Create a link by selecting the desired element and pressing the link button.

 

edit/delete

 

Edit or remove a link by selecting an established link and pressing the link button.
