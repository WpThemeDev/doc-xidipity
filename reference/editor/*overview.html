The Xidipity classic editor has two modes of operation; visual and text. The visual mode is where content is created in a manner which closely resembles the result as viewed on the web. The text mode presents the underlying HTML code the browser uses to render the page. It is important to note the editor compresses the HTML code to increase the speed of execution so much of the formatting is lost. Switching between modes is accomplished by clicking the visual or text tab on the right.

 

The expectation is most of the author's time will be spent using the visual mode.

 

The "Add Media" button located in the top left of the toolbar region is a core WordPress function which explains why it is not included in the toolbar. It's purpose is to place an image or video (media library/web) into your content. In many ways is operates much like an Xidipity template.
