Description:Classes for controlling alignment attributes.

Class Function
aln:left Horizontal align block left
aln:center Horizontal align block center
aln:right Horizontal align block right
aln:baseline Vertical align baseline
aln:top Vertical align top
aln:middle Vertical align middle
aln:bottom Vertical align bottom
aln:sub Vertical align sub
aln:sup Vertical align sup
aln:text-top Vertical align text top
aln:text-bottom Vertical align text bottom
aln:text-left Horizontal align text left
aln:text-center Horizontal align text center
aln:text-right Horizontal align text right
aln:text-justify Justify text left & right

 

property: aln

property value:

/* all media */

/* standard media */
