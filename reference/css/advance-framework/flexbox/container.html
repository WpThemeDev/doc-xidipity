Description:Classes for controlling flexbox container attributes.

Class Function
fx:c Flexbox column container /wo wrap
fx:c-rev Flexbox reversed column container /wo wrap
fx:cw Flexbox column container /w wrap
fx:cw-rev Flexbox column container /w wrap reversed
fx:r Flexbox row container /wo wrap
fx:r-rev Flexbox reversed row container /wo wrap
fx:rw Flexbox row container /w wrap
fx:rw-rev Flexbox row container /w wrap reversed
