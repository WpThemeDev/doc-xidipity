Description:Classes for controlling flexbox child (item) attributes.

Class Function
fxd:1 0 0 auto
fxd:2 0 1 auto
fxd:3 1 0 auto

 

property: fxd

property value:

1 (0 0 auto)

 

Set flex-grow to 0, flex-shrink to 0, flex-basis to auto

 

/* all media */
<div class="fxd:1 …">
/* standard media */
<div class="sm)fxd:1 …">

 

2 (0 1 auto)

 

Set flex-grow to 0, flex-shrink to 1, flex-basis to auto

 

/* all media */
<div class="fxd:2 …">
/* standard media */
<div class="sm)fxd:2 …">

 

3 (1 0 auto)

 

Set flex-grow to 1, flex-shrink to 0, flex-basis to auto

 

/* all media */
<div class="fxd:3 …">
/* standard media */
<div class="sm)fxd:3 …">

 

​ Item order is not included as a property of this element as it is set to 0 which means no order preference. Auto refers to the flex basis (dimension) of the child and can be set if required by using the height and width properties.
