Description:Classes for controlling cursor style.

Class Function
cur:auto Auto select style
cur:default Cursor to default style
cur:pointer Cursor to pointer style
cur:text Cursor to text style
cur:wait Cursor to wait style
Hover Event
cur:auto!hover Auto select on hover event
cur:default!hover Cursor to default on hover event
cur:pointer!hover Cursor to pointer on hover event
cur:text!hover Cursor to text on hover event
cur:wait!hover Cursor to wait/busy on hover event
Focus Event
cur:auto!focus Auto select on focus event
cur:default!focus Cursor to default on focus event
cur:pointer!focus Cursor to pointer on focus event
cur:text!focus Cursor to text on focus event
cur:wait!focus Cursor to wait/busy on focus event

property: cur
