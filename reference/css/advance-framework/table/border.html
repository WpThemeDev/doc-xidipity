Description:Classes for controlling table border attributes.

Class Function
bdr:collapse Share table and cell borders
bdr:separate Separate table and cell borders
bdr:full Show all table borders
bdr:hidden Show no table borders
bdr:space+0 No spacing between table borders
bdr:space+0.25 Set table border spacing @ +0.25rem
bdr:space+0.5 Set table border spacing @ +0.5rem
bdr:space+0.75 Set table border spacing @ +0.75rem
bdr:space+1 Set table border spacing +@ 1rem
bdr:space+1.25 Set table border spacing @ +1.25rem
bdr:space+1.5 Set table border spacing @ +1.5rem
bdr:space+1.75 Set table border spacing @ +1.75rem
bdr:space+2 Set table border spacing @ +2rem
