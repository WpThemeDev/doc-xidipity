Description:Classes for controlling font size attributes.

Class Function
fnt:size-3x-large Set font to predefined xxx-large
fnt:size-2x-large Set font to predefined xx-large
fnt:size-x-large Set font to predefined x-large
fnt:size-large Set font to predefined large
fnt:size-x-medium Set font to predefined x-medium
fnt:size-medium Set font to predefined medium
fnt:size-small Set font to predefined small
fnt:size-x-small Set font to predefined x-small
fnt:size-2x-small Set font to predefined 2x-small
fnt:size-larger Step font larger
fnt:size-smaller Step font smaller

 

property: fnt

property value:

size-3x-large
