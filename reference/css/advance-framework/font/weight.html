Description:Classes for controlling font weight attributes.

Class Function
fnt:weight-bold Set font weight bold
fnt:weight-bolder Set font weight bolder
fnt:weight-lighter Set font weight lighter
fnt:weight-normal Set font weight to normal
fnt:weight-default Set font weight to default

 

property: fnt

property value:

weight-bold

 

 Xidipity WordPress Theme 

 

/* all media */
<div class="fnt:weight-bold …">
/* hover event */
<div class="fnt:weight-bold!hover …">
/* focus event */
<div class="fnt:weight-bold!focus …">
