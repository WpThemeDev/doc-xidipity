Description:Classes for controlling font family attributes.

Class Function

fnt:family-serif Set font to serif as defined
fnt:family-mono Set font to mono as defined
fnt:family-condensed Set font to condensed as defined
fnt:family-cursive Set font to cursive as defined
fnt:family-fancy Set font to fancy as defined
