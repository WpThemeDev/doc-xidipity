Description:Classes for controlling the bottom margin. An element's margin is the space between its border and other content.

Class Function
mar:bottom+0 No bottom margin
mar:bottom+0.25 Bottom margin @ +0.25rem

mar:bottom+1.25 Bottom margin @ +1.25rem

mar:bottom+1.75 Bottom margin @ +1.75rem
mar:bottom+2 Bottom margin @ +2rem
mar:bottom-0 No bottom margin

mar:bottom-0.75 Bottom margin @ -0.75rem
