Description:Classes for controlling padding on all sides (default). An element's padding is the space between its content and its border.

Class Function
pad:+0 No padding
pad:+0.25 Pad @ 0.25rem
pad:+0.5 Pad @ 0.5rem
pad:+0.75 Pad @ 0.75rem
pad:+1 Pad @ 1rem
pad:+1.25 Pad @ 1.25
pad:+1.5 Pad @ 1.5
pad:+1.75 Pad @ 1.75rem
pad:+2 Pad @ 2rem

 

​ All is implied in the property and does not have to be specified.

 

property: pad

property value:

+0.25 to +2 (rem)
