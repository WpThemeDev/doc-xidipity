template markers

#url#

Link to execute when button is clicked.

#img#

Fully formatted image element. The example above uses an image sized 128x128px. The SVG icon example uses an embedded svg icon converted to Base64.

#fa-icon#

Fully formated font awesome icon element ( <i class="fa attributes">&#x200B;</i> ).

#text#

Button text.

 

​ Template markers must be updated before use.
