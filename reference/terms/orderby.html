definition

The column used to sort database results.

 

Column Description
none No order
ID Order by post ID
author Order by author
title Order by title
name Order by post name (post slug)
date Order by date (default)
modified Order by last date modified
parent Order by post/page parent
rand Random order
comment_count Order by number of comments
menu_order Order by hierarchical page order
title_meny_order Order by both menu_order AND title.
