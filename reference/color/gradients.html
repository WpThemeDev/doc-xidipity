Description:Classes for displaying a predefined gradient backgrounds.

Class Function
grd:pri-bright Display bright primary hue background gradient
grd:pri Display primary hue background gradient
grd:pri-dark Display dark primary hue background gradient
grd:sec-bright Display bright secondary hue background gradient
grd:sec Display secondary hue background gradient
grd:pri-dark Display dark secondary hue background gradient
grd:tint-bright Display bright tint hue background gradient
grd:sec Display tint hue background gradient
grd:pri-dark Display dark tint hue background gradient
