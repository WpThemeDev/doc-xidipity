about

 

The Xidipity hamburger menu is a replacement for the WordPress drop down menu. It begins life as a standard page and there are no restriction on content with the except the design should support mobile as well as larger screens. The benefit of using this solution is it gives the author complete control over the layout and when leveraged with Xidipity templates the solution is very powerful. To activate, create the desired menu page and name the permalink (slug) "xidipity-toc". To remove the page title from the final result use the "no-title" page template.
