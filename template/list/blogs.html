markers

#class#

Optional list class(s). Typical options include:

  • no-bullets = no bullets
  • li:qtr-spaced = quarter space list items
  • li:dbl-spaced = double space list items
  • li:tpl-spaced = triple space list items
  • misc = background hue, top and bottom margins, etc

 

#bullet#

Optional font awesome icon.

 

#orderby#

How items will be grouped. See terms, orderby for more information.

 

#order#

Order of the gallery (a/d). See terms, order for more information.

 

#xclude#

Optional media categories to exclude.

 

#cat#

Media categories to include.
