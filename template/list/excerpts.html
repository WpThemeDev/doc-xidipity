markers

#orderby#

How items will be grouped. See terms, orderby for more information.

 

#order#

Order of the gallery (a/d). See terms, order for more information.

 

#align#

Featured image alignment. "0" for all.

 

#count#

Optional number of rows to display. "0" for all.

 

#count#

Optional "page" the display if total rows exceeds count.

 

#xclude#

Optional categories to exclude.

 

#cat#

Categories to include.
