Description:Submit button wrapped in a form.

 

standard

 

 

<!-- xwpt: 200208/tmpl/btn/dft              -->
<form action="#url#">
    <button type="submit">#content#</button>
</form>
<!-- /xwpt: 200208/tmpl/btn/dft             -->

 

alternate

 

 

<!-- xwpt: 200208/tmpl/btn/dft              -->
<form action="#url#">
    <button class="bg:blk bg:bas-800!hover" type="submit">#content#</button>
</form>
<!-- /xwpt: 200208/tmpl/btn/dft             -->

 

template markers

#url#

Link to execute when the button is pressed.

 

#content#

Button content. The content may consist of the following:

  • text only
  • text and icon
    • font awesome icon
    • google icon
  • image ( png/jpg scaled to appropriate dimensions )
  • animated gif ( scaled to appropriate dimensions )
  • emoji / unicode character

 

​ Template markers must be updated before use.
