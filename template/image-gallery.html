markers

#class#

Optional image class(s). See customization, image transformation for options.

 

#align#

Optional caption alignment.

  • l = left
  • c = center
  • r = right

 

#desc#

Display description.

  • n = no
  • y = center

 

#cols#

Number of columns (2/3/4)

 

#ordby#

How items will be grouped. See terms, orderby for more information.

 

#order#

Order of the gallery (a/d). See terms, order for more information.

 

#xclude#

Optional media categories to exclude.

 

#cat#

Media categories to include.
