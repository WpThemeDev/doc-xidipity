markers

⩹

The template can be installed as early bound or late bound. Using the early bound method negates the need for the end of content marker (⩹ ecm) and it can be deleted if desired. Otherwise, install the template as late bound without modification. The purpose of the ecm is to assist the editor manage white space and it is suggested adding or deleting lines be performed with the cursor next to the marker. It will not display on the web. To avoid the whitespace issue, create your content outside of the container and then cut/pasted into the container.
