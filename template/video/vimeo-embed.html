markers

#title#

Video title/description.

 

#iframe#

Vimeo iframe. To obtain the iframe goto Vimeo (browser) and select the desired video. Click share, and then embed. To the right is a link titled "show options". Click this link to set display preferences. When all changes are in place click inside the text area to select and copy the contents to the clipboard. Highlight the #iframe# marker and paste the contents of the clipboard. The solution is responsive.

 

​

Early bound template. Markers must be updated before use.

The editor in visual mode may reformat the code of the embedded iframe in a manner which changes the desired result. In those circumstances the template will have to be added to the source html in text mode and saved without review in visual mode.
