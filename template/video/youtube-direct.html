markers

#title#

Video title/description.

 

#id#

YouTube video ID. To obtain the ID goto YouTube (browser) and select the desired video. Click share and the URL will be displayed with the ID as an eleven digit code at the end. In the case of this example the share URL presented is (https://youtu.be/gn5QmllRCn4).

 

#poster#

Poster image URL. There are no restrictions on this image and the width and height parameters are optional.
