template markers

Marker Description
#CLASS# List class.
 

Typical options include:

  • no-bullets= no bullets
  • li:qtr-spaced= quarter space list items
  • li:dbl-spaced= double space list items
  • li:tpl-spaced= triple space list items
  • other= background hue, top and bottom margins, etc
#BULLET# Optional font awesome icon.
#DEPTH# How levels of the tree to display (0=all).
#TITLE#

Beginning title and/or titles(s) to exclude. For example, assuming the following page hierarchy:

  • Paper Backs
    • Science Fiction
    • Biographies
    • Romance
    • Travel
  • Hard Cover

To show a pages beginning at "Paper Backs" that exclude "Romance" would be executed by "Paper Backs,-Romance". Although there can only be one starting point, there can be multiple exclusions.
  ​Multiple titles should be separated by a comma.

​The template code is available for viewing on a larger display.

​ Template must be configured before use.
