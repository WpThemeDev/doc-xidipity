template markers

Marker Description
#CLASS# Optional image class(s). See customization, image transformation for options.
#CAPTION# Caption alignment/display.
 
  • l= left
  • c= center
  • r= right
  • x= no caption
#CONTENT# Display content.
 
  • n= no
  • y= yes
#ORDBY# How items will be grouped. See terms, orderby for more information.
#ORDER# Order of the gallery (a/d). See terms, order for more information.
#CATEGORY# Media library category filter.
  ​Multiple categories should be separated by a comma.

​The template code is available for viewing on a larger display.

​ Template must be configured before use.
