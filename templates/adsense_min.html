template markers

Marker Description
#TITLE# Title / description.
#SCRIPT# Adsense script from the Google adsense dashboard. Typically this is javascript wrapped in "<script></script>" tags.

​The template code is available for viewing on a larger display.

sidebar widget

  1. Open the widget panel from the admin menu (appearance ​ widgets)
  2. Drag the "Custom HTML" widget to the sidebar panel
  3. Title as desired
  4. In the code editor, paste your responsive ad unit script
  5. Press Save

disclaimer

The Google Adsense script evaluates your page to determine the best ad to fill the designated space. It is important to keep in mind the solution is good but it is not 100%. Generally speaking, the responsive ad unit works well as a sidebar widget and the in-article ad unit for content ads. In some cases it may be necessary to set the responsive property "data-full-width-responsive" to false to achieve the desired result.

​ Template must be configured before use.
