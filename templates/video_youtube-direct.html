installation

  1. Upload desired image to the media library
  2. Place the image in your content using the Add Media button
  3. Single click image and select edit ( ​ )
  4. Under DISPLAY SETTINGS
    • Select "Custom URL" from the "link to" dropdown button
    • Paste/enter the YouTube direct link
  5. Click Update

YouTube direct link

The syntax of the YouTube direct link is as follows:

<a href="https://www.youtube.com/embed/ #ID# ?autoplay=1&amp;amp;start=0">

To obtain the #ID#, go to YouTube (browser) and select the desired video. Click share and the URL will be displayed with the ID as an eleven digit code at the end. In the case of this example the share URL presented is ( https://youtu.be/gn5QmllRCn4 ).

​ Template must be configured before use.
