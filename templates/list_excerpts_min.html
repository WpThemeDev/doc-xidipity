template markers

Marker Description
#ALIGN_IMG# Featured image placement.
 
  • l= left
  • r= right
  • x= no image
##CNT## Maximum number of posts to display (0=all).
#ORDBY# How items will be grouped. See terms, orderby for more information.
#ORDER# Order of the list (a/d). See terms, order for more information.
#CATEGORY#

Category(s) to include and/or category(s) to exclude. For example, assuming the following categories:

  • Paper Backs
    • Science Fiction
    • Biographies
    • Romance
    • Travel

To show excerpts of "Paper Backs" that exclude "Romance" would be executed by "Paper Backs,-Romance".
  ​Multiple categories should be separated by a comma.

​The template code is available for viewing on a larger display.

​ Template must be configured before use.
