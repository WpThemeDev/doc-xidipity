template markers

Marker Description
#CLASS# List class.
 

Typical options include:

  • no-bullets= no bullets
  • li:qtr-spaced= quarter space list items
  • li:dbl-spaced= double space list items
  • li:tpl-spaced= triple space list items
  • other= background hue, top and bottom margins, etc
#BULLET# Optional font awesome icon.
#DEPTH# Optional hierarchical depth. "0" for all.
#ACTIVE# Only show categories which have posts assigned to them.
#CATEGORY#

Beginning category and/or category(s) to exclude. For example, assuming the following category hierarchy:

  • Paper Backs
    • Science Fiction
    • Biographies
    • Romance
    • Travel

To show a category list beginning at "Paper Backs" that exclude "Romance" would be executed by "Paper Backs,-Romance". Although there can only be one starting point, there can be multiple exclusions.
  ​Multiple categories should be separated by a comma.

​The template code is available for viewing on a larger display.
